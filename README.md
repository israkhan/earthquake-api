# Earthquake API

## Getting Started

Git clone this repository and run the following:

```
cd functions
npm install
```

## Development

To serve any changes locally:

```
npm run server
```

If making any changes to API, please make sure to add and/or udpate swagger docs.

## Documentation

The documentation of this API is generated by swagger. In order to view documentation, run `npm run server` and navigate to `http://localhost:5001/earthquake-notification-59115/us-central1/app/api-docs/`

## Deployment

This API is hosted on Firebase. In order to deploy changes, please reach out to the owner for Firebase credentials.

Once credentials are obtained, run

```
npm run deploy
```
